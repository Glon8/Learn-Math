# Learn math:

Its my graduation project in Atid college. It ment to be used by pupils and children, to exercise math and learn some cool tricks with.
Build on React-Vite, Mongo Atlas, JS. The site also includes an "online teacher" for kids, which is a Gemini, it very linear toward math, so if one of the users ll have trouble with excercises, he/she could ask someone about. 

**Link to:** https://learn-math-now.netlify.app/

**To run locally:**

download files, and run each part seperately:
npm run dev - for frontend
npm test - for backend

**Connect by link:** http://localhost:5174/
